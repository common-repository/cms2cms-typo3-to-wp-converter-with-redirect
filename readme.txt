Plugin for automated data migration from TYPO3 to WordPress. Convert TYPO3 to WordPress easily and fast.

== Description ==
You’ve been running a TYPO3 website for a while. Now you want to try out something new, and decided it to be WordPress. What will happen to my current  TYPO3 website content though? How to move it to WordPress? Here’s a plugin to help you answer the concerning questions.

= HOW THIS PLUGIN HELPS: =
1. Launching Free Demo Migration helps to preview the migration results first;
2. Average migration time is from several minutes to several hours, depending on how many content items you’re up to move from TYPO3 to WordPress;
3. Migrate the content you want. Map the data and choose how the migrated content lives and looks on WordPress;
4. Complete migration price depends on the amount of content you’re going to move from TYPO3 to WordPress.

= WHAT WILL BE MOVED FROM TYPO3 TO WORDPRESS? =
* Pages along with the content included there;
  Attachments (images and videos);
  Users, and their roles;
  Categories;
  Images included in the content body.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Automated 301 redirects from existing TYPO3 site to the new WordPress website.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
* Our support team is available live 24/7.
* For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Typo3 To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide the URL address of your Existing website.
7. Download the Connection Bridge and place it into the Existing website root folder (where index.php file is located). You can find a detailed instruction here.
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
